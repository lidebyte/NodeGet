# 一键部署

一键部署，就是字面所讲，通过一次点击直接部署

当你看到类似下面的按钮时，说明主题适配了 NodeGet 规范主题框架要求，可以快捷部署到控制面板上。

<a href="https://dash.nodeget.com/#/dashboard/theme-management?add=https://nodeget.pages.dev">
  <img src="https://dash.nodeget.com/deploy-button.png" alt="deploy button" width="230px" />
</a>

点开后的界面如图所示：
![DvFKfbr72YB76471csYxJ5qdarfs84Xm.webp](https://cdn.nodeimage.com/i/DvFKfbr72YB76471csYxJ5qdarfs84Xm.webp)

安装过程会自动启用主题，创建token，全程不用用户操心

在主题管理功能里面有很多辅助功能，不限于：

- 从本地文件夹/压缩包安装及更新
- 从远程安装及更新
- 利用主题提供的快捷表单来调整主题配置选项
- 保存和使用token预设
- 调整自定义样式和js
- 下载当前的主题文件，可以将下载的主题部署到cloudflare pages上
