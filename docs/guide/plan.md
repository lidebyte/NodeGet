# 开发计划

NodeGet 有更多的开发计划，未来会持续更新

欢迎感兴趣的开发者 fork & pr，为项目贡献代码


## 文件系统

计划支持一组 WebDAV 改造后的文件系统 API

## More Hooks

支持更多的生命周期钩子函数

## 补充常用插件

通过插件扩展，来支持一些基本的需求，比如：

- Docker 管理
- 进程管理
- 防火墙管理

也欢迎开发者实现后通知我们，如质量较高可以作为官方推荐方案

## iOS & Android 应用

欢迎开发者适配

## Binary API Client

除了浏览器 Dashboard，还考虑支持二进制 API 客户端，扩展下操作工具

大概率是非交互式（命令行软件），也可能额外支持交互式 TUI 工具

## Js Worker for Agent

计划在 Agent 上支持 Js Worker，可能是外置式，还需要讨论

## 扩展商店

可能会以第三方应用集合的形式发布，待定，也欢迎第三方方案

用于收集用户创建的主题/Worker/扩展

## Server-Server 通信

主控间的数据通信协议

## 时序数据库
可能会支持时序数据库

## 官方主控托管服务
等程序迭代的稳定了后，也许会有
