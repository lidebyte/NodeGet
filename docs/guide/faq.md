# 常见问题

## token/密码忘记了怎么办

先用命令行重置token，然后在面板上修改主控节点的token，最后修改所有系统自动安装的worker的env里面的token变量


## 我可以自托管dashboard吗
完全可以，NodeGet的所有代码都是开源的，包括dashboard部分
但是建议直接使用官方的dashboard网址，以保持和上游稳定版本的同步更新，减少碎片化问题
如果认为有必要自建dashboard部分，也完全支持

## 如何添加新的延迟曲线
在定时任务里面增加ping/tcp_ping任务，或者编辑某个定时任务增加新的agent节点

## 如何迁移
如果不保留历史数据，首选多主架构，增加一个主控，然后删掉原主控即可

如果保留数据，需要重新安装，关闭新装主控，复制原数据库和配置文件到新安装的机器，重启主控，修改被控端地址
